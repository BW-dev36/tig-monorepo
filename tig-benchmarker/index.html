TIG Browser Benchmarker

Instructions:

  1. Open the browser console
  2. Setup: `await benchmarker.setup(<api_url>, <api_key>, <player_id>)`
  3. Select Algorithm: `await benchmarker.select_algorithm(<challenge_id>, <algorithm_id>)`
  4. Start: `await benchmarker.start(<num_workers>, <ms_per_benchmark>)`
  5. View State: `await benchmarker.state()`
  6. Stop: `await benchmarker.stop()`

Notes:

  • challenge_id for satisfiability: c001
  • challenge_id for vehicle_routing: c002
  • challenge_id for knapsack: c003
  • algorithm_id format cXXX_aXXX, e.g. first algorithm for satisifaiblity is c001_a001
  • recommended ms_per_benchmark is 15000 (15s)
